# Project: Caroline — Roadmap

This is a living document of planned features and long-term goals. Order within each section reflects rough priority, not a committed release schedule.

## Coming Soon

### Discord Messaging
Send and receive Discord messages directly from the kiosk — no need to set up a dedicated channel. Chat with Caroline and have her relay messages through your existing Discord account.

### Telegram Support
Connect a Telegram bot so Caroline can send and receive messages, alerts, and reminders through Telegram.

### Outlook / Microsoft 365 Calendar
Calendar read, add, and delete support for Microsoft 365 and Outlook accounts, alongside the existing Google Calendar integration.

### Nanoleaf Lights and Panels
Smart lighting control for Nanoleaf panels and shapes, joining the existing Philips Hue integration.

### Mobile Polish
Caroline now has a first-pass phone/tablet browser layout for quick local chat and controls. Next steps are polish: better compact settings, tighter touch targets, and more graceful handling of optional widgets on small screens.

### More Avatars
Additional built-in avatar options. Original character art and expressive animations are the long-term goal — see [Calling Artists and Animators](../README.md#calling-artists-and-animators).

---

## Long-Term Roadmap

### Cross-Device Session Sync
Shared conversation history across all connected kiosks and browsers, so every screen picks up where the last one left off. Includes a per-device privacy guard — devices can opt out of syncing and keep history local only.

### Caroline-to-Caroline Communication
Instances of Caroline on different networks or households can exchange messages and share context, with user consent. Early thinking: a relay-based approach that keeps messages private and does not require a central server.
