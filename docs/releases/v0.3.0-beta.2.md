# Project Caroline v0.3.0-beta.2

Public beta hotfix release.

## Install This Exact Build

```bash
curl -fsSL https://raw.githubusercontent.com/Project-Caroline/project-caroline/v0.3.0-beta.2/install.sh | bash -s -- --channel v0.3.0-beta.2
```

## Changes Since v0.3.0-beta.1

- Fixed calendar event title cleanup so requests like "add Eat dinner to my calendar at 9pm" create "Eat dinner" instead of including the calendar phrase in the title.
- Handles common misspellings such as `calender` and `caliender`.
- Applies the cleanup to both direct local parsing and AI-generated calendar action JSON.
- Updated local AI guidance: best experience is OpenRouter; recommended local quality is `qwen2.5:1.5b`; fast local fallback is `qwen2.5:0.5b`; safe legacy local choice is `gemma3:1b`.
- Added hardware-aware installer recommendations so local Ollama setup preselects a model based on platform and RAM.
- Local Ollama requests now disable model thinking mode to avoid empty visible replies from thinking-first models such as `gemma4:e2b`.
- Added an experimental SteamOS home-directory installer for Steam Deck testing without disabling SteamOS read-only mode.
