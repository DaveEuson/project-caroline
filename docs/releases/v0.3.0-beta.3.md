# Project: Caroline v0.3.0-beta.3

Public beta companion release.

## Install This Exact Build

```bash
curl -fsSL https://raw.githubusercontent.com/Project-Caroline/project-caroline/v0.3.0-beta.3/install.sh | bash -s -- --channel v0.3.0-beta.3
```

## Changes Since v0.3.0-beta.2

- Added and stabilized the Windows companion client integration path.
- Restored the kiosk companion sync-code pill and made it copyable from the top bar.
- Fixed direct WebSocket `remember ...` commands so they become memory actions.
- Fixed Privacy Mode so memory saves are blocked instead of only hidden in the UI.
- Persisted memory retention, privacy, and companion pairing settings through Node-RED settings storage.
- Added the kiosk update action with clearer progress and status feedback.
- Fixed companion connection URL normalization so LAN host URLs are saved instead of localhost-only values.
- Added optional Carl awake/asleep GIF avatar assets while keeping Caroline as the default avatar.
- Cleaned the Node-RED canvas into labeled groups to make the flow easier to navigate.
- Removed old unreferenced media/screenshot assets from the release tree.
