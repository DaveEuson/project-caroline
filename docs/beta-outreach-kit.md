# Project: Caroline Beta Outreach Kit

Use this when you are ready to ask for a small beta group without turning it into a loud launch.

## Posting Rule

Do not mass-post the same link everywhere.

For public communities, post only after checking that community's rules. Reddit's own guidance says promotional content is not automatically spam, but repeated or unsolicited mass promotion can be treated as spam, and many communities enforce strict self-promotion rules.

Helpful references:

- Reddit spam policy: https://support.reddithelp.com/hc/articles/360043504051-Spam
- Reddit moderator guidance on promotional content: https://support.reddithelp.com/hc/en-us/articles/28012014962580-How-do-I-keep-spam-out-of-my-community

## Best First Targets

Start with warm and semi-warm audiences before public communities.

| Priority | Place | Why | Post Type |
|---|---|---|---|
| 1 | Personal LinkedIn | People know Dave and will be kinder with beta feedback | Short personal post |
| 2 | GitHub Discussions | Native home for testers, issues, and follow-up | Announcement |
| 3 | Discord or Slack groups Dave already belongs to | Better signal than cold posting | Soft beta ask |
| 4 | Raspberry Pi / Ubuntu / homelab communities | Strong hardware fit | Rule-checked beta request |
| 5 | Steam Deck communities | Good story for Carl, but be careful with self-promo rules | Rule-checked SteamOS test request |
| 6 | Product Hunt / Hacker News / broader launch sites | Better after 5-10 outside tests | Later launch, not first beta |

## GitHub Discussion Draft

Title:

```text
Looking for 10 beta testers for Project: Caroline
```

Body:

```text
Project: Caroline is ready for a small beta round.

I am looking for about 10 people to test the current release on real hardware:

- Raspberry Pi OS Desktop
- Ubuntu Server or Desktop
- Pop!_OS or similar Ubuntu-based desktops
- Steam Deck Desktop Mode
- Browser/client mode from another computer on the same LAN

The goal is not praise. The goal is to find the rough edges:

- Install friction
- Confusing setup steps
- Broken widgets
- Weird AI replies
- Slow local-model behavior
- Anything that looks polished or surprisingly useful

Start here:
https://github.com/Project-Caroline/project-caroline/blob/release/docs/beta-testing.md

Please do not include secrets in feedback. No API keys, OAuth tokens, Discord bot tokens, passwords, or private calendar details.
```

## Do Not Post Yet

Save these for later, after at least 5 completed beta reports:

- Product Hunt
- Hacker News launch/show post
- Larger AI communities
- Paid ads
- Broad Reddit cross-posting
